## Ascii

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/ascii.git
cd ascii
go build -o ascii .
```

### Usage

```bash
./ascii --help
```

---

## Quick Start

```bash
./ascii [command]
```
