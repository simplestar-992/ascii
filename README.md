# ASCII

**Useful CLI tool for developers**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/ascii.git
cd ascii
go build -o ascii .
```

## Usage

```bash
./ascii --help
```
